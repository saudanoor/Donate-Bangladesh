Important FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects.

2. How do you select an element from the DOM?

You can select an element from the DOM using methods like document.querySelector() or document.getElementById().

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements, making event handling more efficient.

4. How do you manipulate an element's attributes and styles using the DOM?

You manipulate an element's attributes using element.setAttribute() or styles using element.style.property = "value".
